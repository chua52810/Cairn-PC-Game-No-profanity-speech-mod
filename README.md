Cairn is an exceptional climbing simulator, boasting stunning visuals, captivating soundscapes, and innovative mechanics. However, there is one significant oversight: the absence of a built-in profanity filter for speech. As of March 2026, official support for this feature remains unavailable. Fortunately, this community modification allows players to adjust the experience, effectively shifting the game's rating from Mature to Family-friendly.

Regardless, since I have time now due to retrenchment, I have decided to tinker, and using a combination of LLMs (Gemini, ChatGpt, Claude) + vibe coding, have been able to come up with a No-profanity speech hack for the game.

So far, have played the game with the hack applied, for about 2 days and have not encounter any profanity in Aava's speech. 

For those interested, you can visit my GitHub for more details.
